Code Smith

Introduction

Are you a programming God? Do you find the challenge of solving algorithmically challenging problems enticing? Then this is the event for you! Solve nerve racking problems, debug seemingly correct code and code your way to be declared the Gods of the trade...

Rules :

• Teams of two shall participate.
• The contest is divided into two rounds: a screening round, and the final round.
• Round I will be a written round comprising of 30-35 questions involving output predictions, debugging and code corrections. Duration: 50 minutes.
• Round II is the final round and this is where you'll actually be coding. 6 questions, 1.5 hours.
• The code will be machine tested and only perfectly correct results will be awarded marks. Partial code will NOT be rewarded.
• Code will be judged on the basis of the time taken and the memory used.
• The programming languages to be used in round II will be C/C++.
• The Operating System will be Debian GNU/Linux and the available compilers will be GCC and Sun Studio.
