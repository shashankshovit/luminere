Introduction

Rules :

Call yourself a geek? Code your way to glory!
1. The event consists of a single round.
2. Duration is 75 minutes.
3. Participants should have sound knowledge of C/C++/JAVA as well as decent knowledge on scripting languages.
4. A set of challenging code snippets will be given.
5. Participants are required to convert into a different language based on the given question.
6. Each team will comprise of maximum two members.

Winning Criteria:-

The teams managing to get maximum points will be declared the winner.
